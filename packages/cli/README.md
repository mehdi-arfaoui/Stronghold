# @stronghold-dr/cli

Open-source disaster recovery automation for cloud infrastructure.

## Quick Start

```bash
# Try without AWS credentials
npx @stronghold-dr/cli demo

# Scan your real infrastructure
npx @stronghold-dr/cli scan --region eu-west-1
npx @stronghold-dr/cli report
```

For full documentation, see the [main repository](https://github.com/mehdi-arfaoui/stronghold).
